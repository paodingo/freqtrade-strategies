Distribution 与 fingerprint

P4A 使用 static, versioned, vendor-friendly manifest。Consumer 只导入所需 component 并生成 project-local lock；升级必须由项目本地 approval 明确触发，禁止 auto-update。

sha256-text-lf-v1：UTF-8/no-BOM；CRLF 与 lone CR 规范化为 LF；除此之外不改 key、whitespace 或 trailing newline；对规范化 bytes 计算 SHA-256。Invalid UTF-8、duplicate JSON keys 和未知 profile fail closed。
